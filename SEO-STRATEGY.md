## PART 9: GITHUB DEPLOYMENT GUIDE

### Connecting to GitHub Pages (Recommended Free Hosting):

1. Create a new repo at github.com: `waterwisekids-com` (or your preferred name)
2. In your terminal (from the repo directory):
```bash
git remote add origin https://github.com/YOUR_USERNAME/waterwisekids-com.git
git push -u origin main
```
3. In GitHub repo Settings → Pages → Source: Deploy from branch → main → /(root)
4. Your site will be live at: `https://YOUR_USERNAME.github.io/waterwisekids-com`
5. Add your custom domain (waterwisekids.com) in Settings → Pages → Custom Domain

### For Custom Domain (waterwisekids.com):
Add these DNS records at your domain registrar:
```
Type: A    Name: @    Value: 185.199.108.153
Type: A    Name: @    Value: 185.199.109.153
Type: A    Name: @    Value: 185.199.110.153
Type: A    Name: @    Value: 185.199.111.153
Type: CNAME Name: www  Value: YOUR_USERNAME.github.io
```

### Future Updates (Every Time You Add Pages):
```bash
git add -A
git commit -m "Add: [description of changes]"
git push origin main
```
GitHub Pages will auto-deploy within ~60 seconds.
